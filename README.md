# Etoolix

## Summary

Etoolix is an online marketplace where users can place unused manufacturing equipment up for sale or rent.

## MVP

### Pages:

- Login Page
- Profile Page
- Home/Explore Page
- For Sale Page
- For Rent Page
- Create Listing Page

### Users will be able to:

- Login/Register
- Enter a new listing for sale
- Enter a new listing for rent

### Database contains:

- Users
- Listings
- Listing URL
- Listing Location
